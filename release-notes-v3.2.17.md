# v3.2.17 发布说明

发布日期：2026-05-15

## 重点更新

- 新增和完善“党员管理”模块，支持党员名册维护、党员详情、党员流动/转入转出记录、任职记录、党员名册导出等日常管理。
- 党员管理中支持对非本村户籍录入的党员进行移出处理，移出后默认不再显示在党员名册中，历史记录仍保留。
- 新增“党费缴纳/党费查缴”功能，支持按年月生成党费记录、查询党员缴费情况、维护缴费基数和实缴金额，并支持党费数据导出。
- 从党员管理列表点击“党费查缴”可直接查看对应党员的缴费情况；重置筛选后再次查询会回到全体党员当月缴费列表。
- 新增“残疾人两项补贴”管理，支持困难残疾人生活补贴、重度残疾人护理补贴的新增、编辑、停发、删除、查询和导出。
- 残疾人管理列表增加补贴入口，可从残疾人档案直接进入对应人员的补贴记录维护。
- 优化安装器安全说明，明确权限配置、数据库、备份目录和本机 `uploads` 文件目录的用途。
- 安装阶段设置目录权限时改为后台写入安装日志，减少多个命令行窗口弹出造成的打扰。
- WebView2 运行时安装改为静默模式，降低首次安装时的额外弹窗。
- 移除运行期隐藏执行 PowerShell 的机器码、硬件绑定和局域网地址采集逻辑，降低杀毒软件误报概率。
- 保护耕地台账重复导入：当导入表未填写“确权情况”时，保留已有地块的确权状态，避免空值覆盖“已确权”等正确数据。
- 优化党费查询底部党支部汇总显示：空党支部显示为“未设置党支部”，无应缴金额时缴费率显示为 `-`。
- 新增正式打包前审查流程，打包时自动执行数据库路径、路由覆盖率、前端构建和 Rust 编译检查。

## 升级说明

- 当前公开版本从 v3.2.15 直接升级到 v3.2.17，v3.2.16 未公开发布，因此本说明包含 v3.2.15 之后累计新增和修复的内容。
- 可直接覆盖安装到原安装目录。
- 正常情况下不会影响现有 SQLite 数据文件、上传文件和数据库备份。
- 如果用户曾在旧版中遇到“隐藏执行 PowerShell”安全软件提示，建议升级到本版本后重新测试安装和首次启动。

## 建议验收项

1. 安装过程中不再弹出多个黑色命令行窗口。
2. 安装安全说明可清楚解释本机权限配置和 `uploads` 目录用途。
3. 启动后杀毒软件不再提示“隐藏执行 PowerShell”。
4. 党员管理可正常新增、编辑、导出，非本村户籍录入党员可按预期移出。
5. 党费查缴可从党员列表跳转查看单个党员，重置后查询可回到全体党员当月列表。
6. 残疾人两项补贴可正常新增、编辑、停发、删除和导出。
7. 局域网授权状态下，主程序仍可正常开放局域网访问。
8. 重复导入耕地台账时，空“确权情况”不会覆盖已有地块确权状态。
