# SmartVillageSystem v3.1.0

发布日期：2026-03-25

## 本版定位

这是面向正式交付的稳定版本，重点收口查询页交互一致性、桌面端启动品牌信息、覆盖安装稳定性，以及旧库异常场景下的管理员登录兜底。

## 本次包含的内容

- 统一多个查询页面的页头字体大小、颜色、表单高度与按钮高度，居民查询保持原样
- 为各查询页面补齐 Enter 回车确认查询，保留原有查询方式不变
- 修复首页统计图标显示异常问题
- 修复程序启动后浏览器标题未及时读取系统设置标题的问题
- 优化覆盖安装流程，降低升级后出现“后端已退出”的风险
- 增加旧数据库权限底座自修复逻辑，异常情况下自动补回 `superadmin / admin123`
- 统一 `superadmin` 默认密码为 `admin123`
- 调整卸载提示文案，明确“保留数据即保留原密码，删除数据才恢复默认账号”
- 清理误入安装包的 `backend/runtime-data` 运行时缓存目录，显著缩小安装包体积

## 升级说明

- 覆盖安装时，如保留原数据目录，则原数据库和原登录密码继续沿用
- 如需恢复默认管理员账号，请删除本地数据后重新安装，默认账号为 `superadmin / admin123`
- 建议正式发布时同步上传安装包、`version.json` 和本更新日志
