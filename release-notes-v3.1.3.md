# SmartVillageSystem v3.1.3

发布日期：2026-04-13

## 本版定位

这是一次面向正式交付的双包发布，主程序与壳客户端都切换到 Tauri/Rust 交付链路，并补齐局域网运行模式的展示一致性。

## 本次包含的内容

- 发布 Tauri/Rust 主程序安装包，支持单机模式与授权后的局域网模式交付
- 发布 Tauri 壳客户端安装包，统一“智慧村居办公系统”品牌文案
- 修复系统信息中局域网授权信息与运行模式展示不一致的问题
- 清理开发态混入的旧 Node 联调认知，发版链路以 Rust/Tauri 为准

## 升级说明

- 覆盖安装到原目录时，数据库目录会保留，原有数据可继续沿用
- 已导入授权的机器可继续以局域网模式运行；未授权机器默认单机模式
- 程序内“检查更新”读取公开仓库 `smat-release` 中的 `version.json`
